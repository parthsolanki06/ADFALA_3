# 💸 Smart Expense Tracker (Flutter + Hive)

## 📌 Project Overview

Smart Expense Tracker is a modern Flutter application designed to help users manage their daily expenses efficiently. The app provides a clean and intuitive interface inspired by fintech applications, allowing users to store, retrieve, update, and delete expense data using local storage.

This project demonstrates the implementation of local data persistence using Hive along with a visually appealing and user-friendly UI.

---

## 🚀 Features

* 💾 Local data storage using Hive
* ➕ Add new expenses with detailed form inputs
* 📋 View all expenses in a structured list
* ✏️ Edit and 🗑️ delete expenses (CRUD operations)
* 🔄 Swipe to delete functionality
* 📊 Analytics screen with graph and category breakdown
* 🧾 Category-based expense organization
* 📅 Date selection for each expense
* 🔁 Data persistence even after app restart
* 🎨 Modern UI with gradient cards and clean layout

---

## 🛠️ Technologies Used

* Flutter (Dart)
* Hive (Local Database)
* fl_chart (for graph visualization)
* Material UI Components

---

## 📱 Application Screens

### 🏠 Dashboard Screen

* Displays total balance
* Lists all expenses
* Swipe to delete functionality
* Floating button to add new expense

### ➕ Add Expense Screen

* Form-based UI for entering:

  * Title / Topic
  * Amount
  * Category
  * Date
  * Income/Expense toggle

### 📊 Analytics Screen

* Graph representing expense trends
* Category-wise breakdown (Food, Travel, Shopping, Health)
* Modern card-based layout with icons/emojis
